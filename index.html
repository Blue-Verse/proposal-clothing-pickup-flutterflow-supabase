현황 분석

  • 현재 운영: 아임웹 호스팅 기반 픽옷(pickot.kr) — 비대면 의류 수거 + 픽포인트 적립 + 픽옷스토어(쇼핑몰) 운영 중
  • 이슈: 아임웹 오류 빈도 ↑, 모바일 앱 부재로 사용자 경험 한계
  • 준비 상태: Figma UI/UX 및 기능 명세 준비 완료, 협업 어드민 개발자 1명 (적극 협조 어려움)
  • 참고 서비스: 리클(모바일 앱 + Refty 쇼핑), 뉴오프(중고 의류 마켓 + 내옷팔기) — 모바일 앱 + 쇼핑 통합 모델 표준
  • 마이그레이션 대상: 회원 데이터, 수거 신청 이력, 포인트 잔액, 주문 내역 (가능 시)

핵심 요구사항 분석

  • 인증: 이메일 + 카카오 소셜 로그인 (Supabase Auth + 카카오 OAuth Provider 또는 Edge Function 토큰 교환)
  • 수거 신청 폼: 날짜·시간·주소·품목·메모 — 주소 검색(다음/카카오 주소 API), 캘린더 + 시간 슬롯 UI, 사진 업로드(Supabase Storage)
  • 상태 추적: 접수 → 수거중 → 완료 등 상태 트랜지션 + 실시간 업데이트(Supabase Realtime) + 알림톡 자동 발송
  • 마이페이지: 신청 히스토리, 픽포인트 내역, 주문 내역, 회원정보 수정
  • 관리자 페이지: 기존 Strapi 활용 또는 Supabase 기반 별도 어드민 (미팅 시 협의)
  • 알림톡 연동: 블룸 AI 채널을 통해 카카오 알림톡 자동 발송 (상태 변경 시 트리거)
  • 쇼핑몰: 카테고리/상품/장바구니/주문/결제(토스페이먼츠) — 픽옷스토어 6종 카테고리 구조 반영
  • 데이터 마이그레이션: 아임웹 → Supabase 회원·주문·포인트 이관 (스키마 매핑 + 검증 스크립트)

제안 포인트

1. Flutter + Supabase 듀얼 백엔드 구축 직접 경험 — Flutter 17K LOC 단일 코드베이스로 iOS+Android 45+ 화면을 Firebase + Supabase 듀얼 스택으로 직접 구현한 경험이 있어, Flutterflow가 생성하는 Flutter 코드 베이스 위에서 Supabase Auth/RLS/Realtime/Edge Functions를 안정적으로 통합 가능합니다.

2. 멀티플랫폼 동시 출시 검증된 패턴 — 동일 코드베이스로 iOS·Android·반응형 웹 3 플랫폼을 4개월 내 동시 출시한 프랜차이즈 관리 앱 경험으로, Flutterflow의 멀티플랫폼 빌드 파이프라인과 스토어 심사 일정을 50일 내 완료할 수 있도록 일정을 설계합니다.

3. 결제 + 예약/신청 + 알림 통합 운영 경험 — 수업 예약·결제·CRM·자동 알림이 결합된 B2C 플랫폼을 풀스코프(기획·디자인·개발)로 출시한 경험으로, 토스페이먼츠 연동·수거 슬롯 예약·카카오 알림톡 트리거를 단일한 상태 머신 기반으로 설계합니다.

4. 포인트/리워드 이코노미 직접 설계 경험 — 7단계 루트박스 + 듀얼 토큰 이코노미를 갖춘 게이미피케이션 앱을 자체 기획·개발한 경험으로, 픽포인트(수거 적립 → 스토어 차감) 트랜잭션 정합성을 보장하는 Supabase 기반 이중 검증 구조를 설계합니다.

5. 산출물 완성도 우선 운영 — 클라이언트 1순위 우선순위인 산출물 완성도에 맞춰, 마이그레이션 검증 스크립트 + 통합 QA + 1개월 무상 하자 보수 + 인수인계 문서까지 포함하여 안정적인 컷오버를 보장합니다.

PRD (Product Requirements Document)

의류 수거 서비스 앱/웹 신규 구축의 기능 요구사항을 정의합니다.

1. 프로젝트 배경 및 목적

현재 아임웹 기반으로 운영 중인 픽옷(pickot.kr) 서비스는 호스팅 안정성 이슈와 모바일 앱 부재로 사용자 경험에 한계가 있습니다. 본 프로젝트는 다음을 목표로 합니다.

  • 아임웹 → Flutterflow + Supabase 독립 플랫폼 마이그레이션으로 안정성 확보
  • iOS·Android 네이티브 앱 + 반응형 웹 동시 출시로 모바일 사용자 경험 개선
  • 수거 신청·상태 추적·포인트·쇼핑몰을 단일 플랫폼에 통합하여 운영 효율화
  • 기존 회원·주문·포인트 데이터를 무손실 이관하여 사용자 영향 최소화
